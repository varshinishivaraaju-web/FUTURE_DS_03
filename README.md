# FUTURE_DS_03
Marketing Funnel & Conversion Analysis – Task 3

1)Objective:
The objective of this project is to analyze marketing funnel performance and identify drop-off points and conversion patterns to improve overall conversion rates.

2)Tools Used:
Power BI
Excel

3)Dataset:
Bank Marketing Dataset (UCI)

4) Key Insights:
i.There is a significant drop-off between total leads and converted customers.
ii.Most leads do not convert, indicating low funnel efficiency.
iii.Conversion performance varies across different customer segments.
iv.Certain customer groups show higher conversion potential.

5)Recommendations:
i.Focus marketing efforts on high-performing segments.
ii.Improve communication and follow-up strategies.
iii.Optimize funnel stages to reduce drop-offs.
iv.Test different campaign strategies to improve conversions.

6)Dashboard:
Power BI dashboard included in this repository
